/* Images Directory */

This folder is for storing all image files used in your portfolio.

## Recommended Images

### Essential Images
- `profile.jpg` - Your profile/avatar photo
- `favicon.ico` - Website favicon (appears in browser tab)
- `screenshot.png` - Portfolio screenshot for sharing

### Project Images (optional)
- `project-1.jpg` - Featured projects
- `project-2.jpg`
- `project-3.jpg`

## Image Optimization Tips

1. **Use the right format:**
   - JPG/JPEG - Photographs and complex images
   - PNG - Graphics and transparent backgrounds
   - WebP - Modern format for smaller file sizes
   - SVG - Vector graphics and icons

2. **Compress images:**
   - Use TinyPNG: https://tinypng.com/
   - Use ImageOptim: https://imageoptim.com/
   - Compress offline: https://www.jpegmini.com/

3. **Recommended sizes:**
   - Profile photo: 300x300px to 500x500px
   - Project thumbnails: 400x300px
   - Hero background: 1920x1080px or larger

4. **File size targets:**
   - Profile photo: < 200KB
   - Project images: < 300KB each
   - Hero image: < 500KB

## Adding Images to Your Portfolio

### For Profile Photo
1. Save your photo as `profile.jpg` in this folder
2. Edit `index.html` to add an image element in the About section (optional)

### For Projects
1. Save project images as `project-1.jpg`, `project-2.jpg`, etc.
2. Add them to the project sections in HTML

## Image Attribution

If using free stock photos, remember to attribute:
- Unsplash: Free use, no attribution required
- Pexels: Free use, no attribution required
- Pixabay: Free use, no attribution required
- Shutterstock: Requires attribution
